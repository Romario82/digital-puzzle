To launch the application, follow the link https://romario82.github.io/d-puzzle/
You need to wait a little while while the browser downloads a mini version of Python in the background and launches the application!

When running locally, you must install Python 3.10 or higher.
If you want to use the graphical interface, install the Flat 0.85.3 framework.

In the config.txt file, use the + symbol to specify the application launch mode (Display GUI, Run in console).
The file name in config.txt is only required for console launch.
